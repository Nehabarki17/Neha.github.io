Projects

AI-Based Real-Time Emotion Analysis System

This project aims to predict and prevent suicidal attempts by using artificial intelligence and healthcare data. The system analyzes patient information and facial expressions to understand emotional states such as sadness, fear, and stress. A Convolutional Neural Network (CNN) is used for emotion detection, while machine learning models calculate a risk score. Based on the risk score, the system classifies individuals into low, medium, or high risk categories. When high risk is detected, the system provides alerts and support recommendations to enable early intervention. This project helps healthcare professionals make accurate, data-driven decisions and supports mental health care through early prediction.

Feedback Management System

The Feedback Management System is a web-based application developed to simplify and automate the process of collecting feedback from students and parents. Instead of using manual forms, users can submit feedback online through a simple and user-friendly interface. The system uses PHP for backend processing to store feedback securely in a database. It implements role-based access control, allowing administrators to log in securely to view, manage, and analyze feedback, while users can only submit feedback. This ensures data privacy and security.The application helps educational institutions save time, reduce paperwork, and improve communication, while providing structured feedback that supports better decision-making and service improvement.
